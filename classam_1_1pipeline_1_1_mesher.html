Detailed Description

The Mesher class represents a base class for meshing algorithms.

Subclasses of Mesher implement specific meshing algorithms to generate meshes from a 3D grid.

Virtual method to build a mesh from a 3D grid that subclasses must implement.

This method is overridden by subclasses to implement specific meshing algorithms. It takes a 3D grid and a map of options as input and returns a pointer to the generated mesh.

Parameters
gridA 3D grid represented as a Mat3D of GridPoint objects.
optsA map containing options as key-value pairs.
Returns
A pointer to the generated Mesh object.

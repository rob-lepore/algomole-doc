Detailed Description

The Validator class provides functionality for validating mesh properties against analytical calculations.

The Validator class provides functionality for validating mesh properties against analytical calculations. The correct analytical values are calculated by calling the UCSF Chimera software. In the project directory will be created a validation folder containing scripts and log files.

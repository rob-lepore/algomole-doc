Detailed Description

The SpaceFiller class represents a base class for space filling algorithms.

Subclasses of SpaceFiller implement specific algorithms for filling 3D space with grid points based on given vector of atoms.

Virtual method to build a 3D grid discretization of space filled with grid points based on atoms' coordinates.

This method is overridden by subclasses to implement specific algorithms for discretizing space with a 3D grid based on a vector of atoms.

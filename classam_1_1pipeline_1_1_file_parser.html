Detailed Description

The FileParser class represents a base class for parsing files containing molecular data.

Subclasses of FileParser implement specific file parsing algorithms to extract data.

Virtual method to parse a file and extract molecular data.

This method is overridden by subclasses to implement specific file parsing algorithms. It takes the file content as a string and a map of options as input and returns a vector containing the parsed data, represented as Atom objects.

Parameters
fileThe content of the file to be parsed.
optsA map containing options as key-value pairs.

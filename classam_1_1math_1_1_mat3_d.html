Detailed Description

template<typename T>
class am::math::Mat3D< T >

The Mat3D class template represents a 3D matrix of arbitrary type.

This class template allows you to create and manipulate a 3D matrix of a specified data type. It provides methods to access elements, query dimensions, and retrieve the data as a vector.

Template Parameters
TThe data type of the elements in the matrix.

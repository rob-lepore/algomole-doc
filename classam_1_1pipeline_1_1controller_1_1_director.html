Detailed Description

The Director class is responsible for directing the construction of SurfaceExtractor objects using the Builder pattern.

This class allows users to specify the type of SurfaceExtractor they want to create (either GAUSSIAN or EDT) and set a SurfaceExtractorBuilder object to construct it.
